Schlag den Prof
===============

How to build
------------

1. Requirements
    1. Java (>= 7)
    2. Apache Maven
    3. bower (with node.js: `npm install bower`)
2. Run `mvn package`.
3. Built JAR file is located in `target` directory.

How to start
------------
1. Requirements
    1. Java (>=7).
    2. Available ports 8024 and 8025.
2. Run `java -jar target/schlagdenprof-*.jar`
3. Open `http://localhost:8024/index.html`.
